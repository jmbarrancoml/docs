Master Freiya's keyboard shortcuts to dramatically speed up your workflow. This reference covers all available shortcuts across the platform.

<Tip>
Shortcuts use `Ctrl` on Windows/Linux and `Cmd` (⌘) on Mac. We'll write `Ctrl/Cmd` to indicate platform-specific keys.
</Tip>

## Global shortcuts

These work anywhere in Freiya:

| Action | Shortcut | Description |
|--------|----------|-------------|
| Quick command palette | `Ctrl/Cmd + K` | Open command search |
| New chat | `Ctrl/Cmd + N` | Start a new AI conversation |
| Add paper | `Ctrl/Cmd + P` | Open Add Paper dialog |
| Search | `Ctrl/Cmd + F` | Focus search box |
| Settings | `Ctrl/Cmd + ,` | Open settings page |
| Toggle dark mode | `Ctrl/Cmd + D` | Switch light/dark theme |
| Toggle sidebar | `Ctrl/Cmd + B` | Show/hide sidebar |
| Close modal | `Esc` | Close any open dialog or modal |
| Help | `?` | Show keyboard shortcuts help |

++IMAGE: **Keyboard shortcuts help overlay showing all global shortcuts in a modal dialog**++

<Note>
The quick command palette (`Ctrl/Cmd + K`) is your fastest way to navigate. Try it!
</Note>

## Navigation shortcuts

Move around the app quickly:

| Action | Shortcut | Description |
|--------|----------|-------------|
| Go to chats | `G` then `C` | Navigate to chats page |
| Go to library | `G` then `L` | Navigate to paper library |
| Go to collections | `G` then `O` | Navigate to collections |
| Go to settings | `G` then `S` | Navigate to settings |
| Previous page | `Alt/Opt + ←` | Browser back |
| Next page | `Alt/Opt + →` | Browser forward |

**How "G then X" works:**
1. Press `G` (releases immediately)
2. Then press the second key
3. Navigation happens

Example: Press `G`, release, press `C` → Goes to Chats

## Platform differences

### Windows/Linux vs Mac

Most shortcuts work the same, but note:

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Main modifier | `Ctrl` | `Cmd` (⌘) |
| Alt modifier | `Alt` | `Option` (⌥) |
| Delete forward | `Delete` | `Fn + Delete` |
| Home | `Home` | `Fn + ←` |
| End | `End` | `Fn + →` |
| Page Up | `Page Up` | `Fn + ↑` |
| Page Down | `Page Down` | `Fn + ↓` |

### Browser-specific notes

**Chrome/Edge:**
- All shortcuts work as documented
- No known conflicts

**Firefox:**
- Some `Ctrl/Cmd + Shift` combos may conflict with browser
- Use browser settings to disable conflicting shortcuts

**Safari:**
- `Cmd + ,` for settings may conflict (native Safari shortcut)
- Use menu navigation instead

## Troubleshooting

<AccordionGroup>
  <Accordion title="Shortcut not working">
    Check:
    - Correct modifier key for your OS (Ctrl vs Cmd)
    - Browser extension isn't intercepting
    - Focus is in the right place (click the area first)
    - Not in text input when using navigation shortcuts
  </Accordion>

  <Accordion title="Conflicts with browser">
    Solutions:
    - Use browser's shortcut customization settings
    - Disable conflicting browser shortcuts
    - Use Freiya in focused mode (F11)
    - Try a different browser
  </Accordion>

  <Accordion title="Can't remember shortcuts">
    Tips:
    - Use `?` to show shortcut help overlay
    - Print this reference page
    - Focus on learning 5 at a time
    - Create flashcards for practice
  </Accordion>

  <Accordion title="Mac keyboard missing keys">
    Mac equivalents:
    - Home: `Fn + ←` or `Cmd + ←`
    - End: `Fn + →` or `Cmd + →`
    - Page Up: `Fn + ↑`
    - Page Down: `Fn + ↓`
    - Delete: `Fn + Delete` (forward delete)
  </Accordion>
</AccordionGroup>
