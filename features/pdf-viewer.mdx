## Overview

Focus mode is designed for serious paper reading and analysis. It combines a full-featured PDF viewer with sophisticated highlighting tools and AI integration.

**Key features:**
- Multi-color highlighting system
- Comments and notes on highlights
- Category-based organization
- Export annotations to markdown
- AI Q&A about the paper
- Vector search integration
- Saved reading sessions

<Info>
Focus mode is available on all tiers, with advanced features on Pro and above.
</Info>

## Accessing Focus mode

### Opening papers in Focus

<Tabs>
  <Tab title="From paper detail">
    1. Navigate to a paper in your library
    2. Click **View PDF** or **Focus** button
    3. PDF opens in Focus mode

    This is the most common way to enter Focus mode.
  </Tab>

  <Tab title="From library">
    1. Hover over a paper card
    2. Click the PDF/Focus icon
    3. Opens directly in Focus mode

    Quick access without viewing paper details first.
  </Tab>

  <Tab title="Recent sessions">
    1. Click **Focus** in the sidebar
    2. View list of recent sessions
    3. Click to resume where you left off

    Pick up right where you stopped reading.
  </Tab>
</Tabs>

### Highlight colors

Freiya provides six highlight colors, each suited for different purposes:

| Color | Suggested Use | Example |
|-------|---------------|---------|
| **Yellow** | Key findings, main results | "Our method achieves 95% accuracy..." |
| **Green** | Methodology, approach | "We used a transformer architecture..." |
| **Blue** | Important quotes for citation | "As Smith et al. demonstrated..." |
| **Red** | Critical points, limitations | "However, this approach has drawbacks..." |
| **Purple** | Questions, future work | "Future research should investigate..." |
| **Orange** | Examples, case studies | "For instance, in the medical domain..." |

<Tip>
Develop a consistent color system and stick to it across all papers for easier review and export.
</Tip>

### Question examples

**Good questions for Focus Q&A:**
```
What methodology did the authors use for data collection?

Summarize the key findings in the Results section.

What are the main limitations mentioned in the paper?

How does this work compare to the Smith 2020 study?

What future research directions do the authors suggest?
```

<Tip>

</Tip>

## Exporting annotations

Save your highlights and notes for use in writing and analysis.

### Export formats

<Tabs>
  <Tab title="Markdown">
    **Best for:** Note-taking apps, writing drafts

    ```markdown
    # Highlights from "Paper Title"

    ## Page 3 - Yellow Highlight
    > Highlighted text goes here...

    **Comment:** My thoughts about this passage
    **Category:** Methodology
    **Date:** 2025-01-15

    ---
    ```
  </Tab>

  <Tab title="CSV">
    **Best for:** Spreadsheets, analysis

### Effective highlighting

**Do:**
- Highlight complete thoughts or sentences
- Add comments explaining why it's important
- Use categories consistently
- Review and refine highlights
- Export for your writing

**Don't:**
- Highlight entire paragraphs (be selective)
- Highlight without reading carefully
- Use colors randomly
- Forget to add comments
- Leave highlights unorganized

## Troubleshooting

<AccordionGroup>
  <Accordion title="PDF won't load in Focus mode">
    Check these issues:
    - PDF URL may be broken or expired
    - File may be too large
    - Network connection problems
    - Try re-adding the paper

    Contact support if issue persists.
  </Accordion>

  <Accordion title="Highlights not saving">
    Possible causes:
    - Network connection lost
    - Browser storage full
    - Account session expired
    - Refresh page and try again
  </Accordion>

  <Accordion title="Can't select text to highlight">
    This happens with:
    - Scanned PDFs (image-based, no text layer)
    - Password-protected PDFs
    - Corrupted PDF files

    Solution: Try re-uploading or use a different PDF source.
  </Accordion>

  <Accordion title="Highlights disappeared">
    Check if:
    - Viewing the correct paper
    - Highlights filter is active (showing none)
    - Paper was re-uploaded (creates new version)
    - Accidentally deleted

    Contact support if highlights are truly lost.
  </Accordion>

  <Accordion title="AI Q&A not working">
    Requirements:
    - Paper must be indexed (automatic on first open)
    - Need active internet connection
    - Must have AI message credits remaining
    - Wait a few seconds for indexing on first use
  </Accordion>
</AccordionGroup>
